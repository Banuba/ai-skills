## List of changes
### NEW Camera layout
The Weatherman mode has received two major improvements. The first is a new camera UI, designed to look better and be more convenient, which means your users will turn to it more often. The second is expanded customization, as now users can add/remove features in the list or change their order to suit individual preferences.

![](attachment:472b57d7-32d4-430f-8fc1-33c8d293eabe:VE_CameraLayout_GIF_1_296x640_.gif)
> ❗ 
Starting from 1.50.0 release all feature are on the right side on camera screen.

### Core dependencies update
This release upgrades the SDK's core technical stack:
- **Target API:** Official support for Android 16 (API 36) is now included.
- **Kotlin:** Compiled with Kotlin 2.2.10
- **Media Engine:** Upgraded to Android Media3 ExoPlayer 1.9.1
- **Koin**: 4.1.1

### Exclude system fonts
New property `useSystemFonts` is added to `MainTextOnVideoTypefaceProvider` to control systems fonts in the list of all fonts for text and caption effects. Default value is `true`

```Swift

{
			...
			
			single<TextOnVideoTypefaceProvider> {
            MainTextOnVideoTypefaceProvider(
                context = get(),
                **useSystemFonts** = false // Disables system fonts
            )
        }
	
}

```

#### Control top Add Sound view
You can control visibility of `Add Sound` view on camera screen for adding audio track. Default value  is`true`
```Swift

{
			...
			
			single<CameraConfig> {
            CameraConfig(
                 ...
                **supportsTopAddAudio** = false, // Hide Add Sound view
            )
        }
	
}
```

#### Fallback to Camera Layout v1
If you still prefer previous camera layout you can switch to using the following config.
```Swift

{
			...
			
			single<CameraConfig> {
            CameraConfig(
                 ...
                **leftContainerFeatures** = cameraV1LeftFeatures, // Import cameraV1LeftFeatures
                **rightContainerFeatures** = cameraV1RightFeatures // Import cameraV1RightFeatures
            )
        }
	
}
```
