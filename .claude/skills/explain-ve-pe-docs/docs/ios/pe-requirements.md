# Banuba Photo Editor SDK Requirements on iOS

Guide contains general requirements for using Photo Editor SDK on iOS.

## License Token

Before you commit to a license, you are free to test all the features of the SDK for free. The trial period lasts 14 days. 

Send us a message to start the [Photo Editor SDK trial](https://www.banuba.com/photo-editor-sdk#form). We will get back to you with the trial token.

Feel free to contact us if you have any questions regarding the [Photo Editor SDK](https://www.banuba.com/support).

## Project settings

This is what you need to run the AR Photo Editor SDK:
- Swift 5.10 or newer,
- Xcode 16.0 or newer,
- iOS 15.0 or newer.
