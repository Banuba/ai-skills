# Installation with Swift Package Manager on iOS

Learn the [SPM Getting Started Guide](https://developer.apple.com/documentation/swift_packages/adding_package_dependencies_to_your_app) if you are new to SPM.

The integration of the PE SDK via SPM is demonstrated in the [spm branch](https://github.com/Banuba/ve-sdk-ios-integration-sample/tree/spm) of our native sample.

Complete the following steps to get the Photo Editor SDK dependencies using SPM:

1. open `App project`  and navigate to the `Package Dependencies` tab;
2. for every package that needs to be installed click the `plus` button and type its repo url;
3. choose `Exact Version` and type the newest SDK version.

The following module must be installed:

| Package name | GitHub link |
|----------|-------------|
| BanubaPhotoEditorSDK-iOS | https://github.com/Banuba/BanubaPhotoEditorSDK-iOS |
