# Customization Options Video Editor SDK on Android  

Banuba Video Editor SDK has built in UI/UX experience and provides a number of customizations you can use to meet your requirements.

**AVAILABLE**

:white_check_mark: Use your branded icons, colors, and text styles. [See details](ve-faq.md)  
:white_check_mark: Localize and change text resources. Default locale is :us:  
:white_check_mark: Make content you want i.e. a number of video with different resolutions  and durations, an audio file.  
:white_check_mark: Masks and filters order. [See details](ve-faq.md)

**NOT AVAILABLE**

:x: Change layout or order of screens after entry point

Feel free to [request](https://www.banuba.com/support) custom UI/UX development.
