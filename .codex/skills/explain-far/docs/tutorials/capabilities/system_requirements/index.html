System Requirements

Supported Platforms​

Mobile

Requirements

  • Any platform: OpenGL ES3.0 and higher
  • Android: supported from Android 8.0. API level 26
  • iOS: supported from iOS 13.0.

Desktop

Requirements PC

  • Support OpenGL 4.3 and up (4.1 for MacOS)
  • Windows 8.1 and up
  • MacOS 10.13 and up
